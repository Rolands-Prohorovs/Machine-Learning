- Most code is in **Jupyter notebooks** (`.ipynb`).  
- Python environment: use the `.venv` (create a new one from `requirements.txt` if available, or install `tensorflow`/`keras`/`pandas`/`numpy` manually).  
- Datasets (CSV files) are included for population analysis.
